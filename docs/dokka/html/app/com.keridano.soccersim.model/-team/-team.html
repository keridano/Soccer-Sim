Parameters

name

the team name

strength

the team strength (used to calculate the matches results). Default value for every team is 50

bonuses

the list of the team bonuses (used in combination with the strength to calculate the matches results)

logo

ID of the res file used for the logo. In order to extend this, this could be an URL to be fetched using Picasso library

wonMatches

number of matches won by the team in the Group stage

lostMatches

number of matches lost by the team in the Group stage

drawMatches

number of matches resulted in a draw in the Group stage

scoredGoals

number of goals scored by the team in the Group stage

concededGoals

number of goals conceded by the team in the Group stage
